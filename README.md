## Features

- **Interactive visualizations** -- quotient space gluing animations, homotopy deformations, covering space unwrapping, metric ball explorers, and more
- **KaTeX math rendering** -- rigorous notation throughout
- **Exercises with hints & solutions** -- from computational to proof-based
- **Progress tracking** -- saved locally in your browser
- **Zero dependencies** -- just open the link and learn

## Quick Start

```bash
# Clone and open locally

open HZ-topology/index.html
```
